== Installation ==
1. Set up an account at KISSmetrics and install the javascript tracker on your site.
2. Copy the entire /kissmetrics/ directory into your /plugins/ directory.
3. Add config settings in plugins/kissmetrics/config/kissmetrics.php
4. Set up cron job as described on http://support.kissmetrics.com/apis/cron
5. Activate the plugin.
6. Don't forget to CHMOD the log directory with the proper permissions
